# Using the System

## The Interface

The main workspace is a conversational interface. You describe what you want in plain language, the system handles everything else. Your conversation history persists so you can reference previous generations, iterate on concepts, and track what's working.

---

## Settings

Before generating, configure your output parameters using the settings bar:

| Setting | Options | Guidance |
|---|---|---|
| **Aspect Ratio** | 4:5 (default), 1:1, 3:4, 9:16, 16:9, 4:3, 3:2 | Use 4:5 for Meta feed. Use 9:16 for Stories/Reels. |
| **Resolution** | 1K, 2K (default), 4K | 2K is optimized for Meta ad delivery. Use 4K for high-production needs. |
| **Output Format** | JPEG (default), PNG | JPEG for smaller file size and faster delivery. PNG for transparency/layering. |
| **Safety Filter** | Low (default), Medium, High | Low block is appropriate for problem-state health/beauty content. |

---

## Downloading Your Creative

Every generated image includes a **Download** button and a **Copy URL** option. Files are named `native-ad-{timestamp}` for clean asset management.
